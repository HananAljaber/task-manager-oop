# مدير المهام باستخدام Python وOOP

مشروع سطر أوامر لإدارة المهام، تم تطويره باستخدام لغة **Python** وتطبيق مبادئ **البرمجة الكائنية (OOP)**.

تم بناء المشروع تدريجيًا، بدءًا من برنامج بسيط يعتمد على القوائم والقواميس، ثم إعادة تنظيمه باستخدام الدوال والملفات وJSON، وبعد ذلك إعادة بنائه باستخدام الكلاسات والكائنات وفصل المسؤوليات.

---

## مميزات المشروع

* إضافة مهمة جديدة.
* تحديد أولوية المهمة:

  * Low
  * Medium
  * High
* عرض جميع المهام.
* تعديل عنوان المهمة.
* تعديل أولوية المهمة.
* تحديد المهمة كمكتملة.
* حذف مهمة.
* البحث عن المهام باستخدام جزء من العنوان.
* عرض إحصائيات المهام.
* حساب نسبة المهام المكتملة.
* حفظ المهام في ملف JSON.
* تحميل المهام تلقائيًا عند تشغيل البرنامج.
* التحقق من صحة مدخلات المستخدم.

## هيكل المشروع

```text
task-manager-oop/
│
├── main.py
├── task.py
├── task_manager.py
├── task_storage.py
├── tasks.json
└── README.md
```

### `main.py`

مسؤول عن:

* عرض القائمة الرئيسية.
* استقبال مدخلات المستخدم.
* استدعاء العمليات المناسبة من `TaskManager`.
* عرض رسائل النجاح والخطأ.

### `task.py`

يحتوي على كلاس `Task` الذي يمثل مهمة واحدة.

بيانات المهمة:

* الرقم التعريفي `id`.
* عنوان المهمة `title`.
* الأولوية `priority`.
* حالة الإنجاز `completed`.

كما يحتوي على العمليات المتعلقة بالمهمة نفسها، مثل:

* إكمال المهمة.
* تعديل العنوان.
* تعديل الأولوية.
* عرض المهمة باستخدام `__str__`.

### `task_manager.py`

يحتوي على كلاس `TaskManager` المسؤول عن إدارة جميع المهام.

من مسؤولياته:

* إضافة المهام.
* البحث عن مهمة باستخدام ID.
* عرض المهام.
* إكمال المهام.
* حذف المهام.
* تعديل العنوان.
* تعديل الأولوية.
* البحث باستخدام العنوان.
* حساب الإحصائيات.
* حفظ التغييرات بعد العمليات.

### `task_storage.py`

يحتوي على كلاس `TaskStorage` المسؤول عن:

* تحويل كائنات `Task` إلى بيانات قابلة للحفظ.
* حفظ البيانات داخل ملف JSON.
* قراءة البيانات من JSON.
* تحويل البيانات المحفوظة إلى كائنات `Task`.

### `tasks.json`

يحتوي على بيانات المهام المحفوظة، حتى لا تختفي المهام بعد إغلاق البرنامج.

## طريقة تشغيل المشروع

تأكد من تثبيت Python 3، ثم حمّل المشروع وانتقل إلى مجلده:

```bash
git clone رابط-المستودع
cd task-manager-oop
```

ثم شغّل الملف الرئيسي:

```bash
python main.py
```

وفي بعض الأنظمة قد تحتاج إلى:

```bash
python3 main.py
```
